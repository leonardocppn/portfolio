Below are some projects, made in my free time, ranging from practical and more or less useful tools to experiments with old and newer technologies:
mbsync mcp server [Repo]
A Model Context Protocol (MCP) server that allows you to locally sync and process your mailbox with the help of an LLM. It was born from the ashes of a project (which I might resume in the future) to manage Thunderbird and was converted to work with mbsync.
Show more...
Main features include:
  • Read emails from Maildir folders synced with mbsync.
  • Search by sender, subject, or content.
  • Filters by date range and read/unread status.
  • IMAP operations: move, delete, archive emails.
  • Batch cleanup: delete/archive multiple emails in a single command.
  • Daily briefing: inbox summary with top senders.
Expense Tracker [Repo]
A PWA designed to help users monitor their personal expenses, categorized by type (essential, leisure, or other) and category (food, health, subscriptions, transport). This allows you to understand, on a monthly basis, the total or specific expenditure, and to clearly identify which areas deserve more attention to save money.
Show more...
This is accompanied by two other trackers: one for daily mood and one for sleep hours. Given that sleep and mood are strongly correlated, this project is also a personal experiment to understand how visible the increase in leisure spending is as an involuntary response to a certain number of consecutive days of bad mood.
As I personally use this PWA (currently collecting data!), the analysis of this correlation will be evaluated and improved in a few months, with more data available.
Book club tracker [Repo] [Demo]
A minimal PWA that allows a small book club to track the reading progress of all its members.
Show more...
Small PWA built to track on mobile the reading progress of a book club made up of me and some friends. To avoid losing touch with others and to stop having to periodically ask: "Where are you up to?".

Frontend written in JavaScript, backend running on Cloudflare Workers with storage on Cloudflare KV. It sends notifications to a shared Discord channel to alert when someone crosses the 50-75-90% threshold of the book.

Minimal, easily extendable/improvable, and hard to recommend for a larger reading group due to the lack of user authentication, which allows anyone to freely falsify other people's data.
Stoic Search [Repo] [Demo]
A minimal browser homepage with a search bar (DuckDuckGo), a bookmarks bar (data saved in LocalStorage), and a different Marcus Aurelius quote for every day of the year to brighten your mood.
Show more...
It uses a simple function to compute the current day of the year (0–365), loads an array of 366 Marcus Aurelius quotes (to handle leap years), and uses the modulo operator to select the corresponding quote: each day shows a different quote following the sequential order of the array, restarting on January 1st of the next year.
